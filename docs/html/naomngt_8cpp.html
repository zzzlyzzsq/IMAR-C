void transferBdd ( std::string  bddName,
std::string  login,
std::string  roboIP,
std::string  password 
)

Transfers the file svm.model, training.means and mapping.txt on the robot Nao.

Parameters:
[in] bddName The name of te BDD to transfer.
[in] login Your user name on the robot.
[in] robotIP The IP adress of the robot.
[in] password The password of the user on the robot.

Training files are sent on the robot Nao via ftp so you have to precise your username on the robot and yout password.

The server ftp must be configured with the following folders: "/data" and "/data/activity_recognition".

Then the program uses the library ftplib to send the training files. It sends mapping.txt, training.means and svm.model which are present in the folder "./bdd/<bdd_name>".
