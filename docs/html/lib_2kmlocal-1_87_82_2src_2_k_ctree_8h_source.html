00029 //----------------------------------------------------------------------
00030 //  kc-tree - the k-center tree.
00031 //      This is a stripped-down modification of the kd-tree of the ANN
00032 //      library (see ann/include/ANN/ANN.h and ann/src/kd_tree.h).
00033 //      There are a number of technical difficulties in attempting to
00034 //      design this structure through inheritance.)  The main difference
00035 //      is that we do not support nearest neighbor searching, but
00036 //      instead support an operation (getNeighbors) which given a set of
00037 //      center points, computes the subset of data points in the tree
00038 //      that is closest to each center.
00039 //
00040 //      In addition to the kd-tree information, the nodes of the kc-tree
00041 //      also store the number of data points associated with each node
00042 //      of the tree and they keep an associated sum and sums of squares
00043 //      for these points.  There are also a few 'hooks' inserted for
00044 //      the eventual goal of extending this to a dynamic structure.
00045 //
00046 //      The tree is constructed in three phases.  The first phase is
00047 //      borrowed from the ANN library, and builds the kc-tree for the
00048 //      data points.  The second phase computes sum and sum of squares
00049 //      for each node, by a simple postorder tree traversal.  The third
00050 //      phase (which may be repeated) is given a set of centers, and
00051 //      computes the candidates for each node in the tree.
00052 //----------------------------------------------------------------------
00053 class KCnode;
00054 typedef KCnode  *KCptr;                 // pointer to kc-node

00113 //----------------------------------------------------------------------
00114 //  Generic kc-tree node
00115 //      Nodes in kc-trees are of two types, splitting nodes which contain
00116 //      splitting information (a splitting hyperplane orthogonal to one
00117 //      of the coordinate axes) and leaf nodes which contain point
00118 //      information (an array of points stored in a bucket).  This is
00119 //      handled by making a generic class kc-node.  The kc-node contains
00120 //      the following basic information.
00121 //
00122 //              n_data          The number of data points associated
00123 //                              with this node.
00124 //              sum             This is the sum of points (i.e., the
00125 //                              weighted centroid) of all the points
00126 //                              associated with this node.
00127 //              sumSq           This is the sum of squares (i.e., the
00128 //                              sum of the dot products of each point
00129 //                              with itself).
00130 //              bnd_box         Bounding box for the cell.
00131 //
00132 //      NOTE: The constructor does no (interesting) initialization.
00133 //      This is handled in getSums().
00134 //----------------------------------------------------------------------
00135

00233 //      Splitting nodes contain a cutting dimension and a cutting value.
00234 //      These indicate the axis-parellel plane which subdivide the
00235 //      box for this node.  The extent of the bounding box along the
00236 //      cutting dimension is maintained (this is used to speed up point

00250 public:
